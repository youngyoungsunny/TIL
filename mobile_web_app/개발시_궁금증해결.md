### 리액트 프로젝트 생성
~~~
terminal
>>>npx create-react-app 프로젝트 이름
//npx: 라이브러리 설치를 도와주는 명령어
//create-react-app: react setting이 다 된 boilerpalte 만들기 쉽게 도와주는 라이브러리

>>success Uninstalled packages. //설치 완료 표시

//그 후 하위 오픈 폴더 연 후, App.js(메인페이지)에서 코딩 시작하기.
